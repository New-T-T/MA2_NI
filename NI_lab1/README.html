1. Installation of brian2 and JupyterLab

We will use Brian2 simulator and JupyterLab to simulate the spiking neurons.

1) Method 1: anaconda environment

  • Create a virtual environment from environment.yml file conda
conda env create -f environment.yml
  • Activate the environment that just installed dependencies
conda activate brian2

1) Method 2: if you don't use conda, please follow the installation instructions of brian2 and JupyterLab
